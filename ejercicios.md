## 1. EQUILIBRIO REDOX - Ejercicios

### EJERCICIO 1: Identificar oxidación y reducción

**Problema**: En la siguiente reacción, identifica qué se oxida y qué se reduce:

**Zn + Cu²⁺ → Zn²⁺ + Cu**

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Identificar los números de oxidación inicial y final

- **Zn**: empieza en 0 (elemento puro) → termina en +2
- **Cu²⁺**: empieza en +2 → termina en 0 (elemento puro)

**Paso 2**: Determinar qué proceso ocurre

- **Zn → Zn²⁺**: PIERDE 2 electrones → **SE OXIDA**
- **Cu²⁺ → Cu**: GANA 2 electrones → **SE REDUCE**

**Paso 3**: Identificar los agentes

- **Agente REDUCTOR**: Zn (causa la reducción del Cu²⁺)
- **Agente OXIDANTE**: Cu²⁺ (causa la oxidación del Zn)

**RESPUESTA**: 
- El zinc se oxida (pierde electrones)
- El cobre se reduce (gana electrones)

### EJERCICIO 2: Balancear ecuación redox en medio ácido

**Problema**: Balancea la siguiente reacción redox en medio ácido:

**MnO₄⁻ + Fe²⁺ → Mn²⁺ + Fe³⁺**

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Escribir las semi-reacciones

**Reducción**: MnO₄⁻ → Mn²⁺
**Oxidación**: Fe²⁺ → Fe³⁺

**Paso 2**: Balancear átomos (excepto H y O)

**Reducción**: MnO₄⁻ → Mn²⁺ ✓ (ya está balanceado el Mn)
**Oxidación**: Fe²⁺ → Fe³⁺ ✓ (ya está balanceado el Fe)

**Paso 3**: Balancear oxígenos con H₂O

**Reducción**: MnO₄⁻ → Mn²⁺ + 4H₂O
- Hay 4 oxígenos en el MnO₄⁻, necesitamos 4 H₂O del lado derecho

**Oxidación**: Fe²⁺ → Fe³⁺ (no tiene oxígenos)

**Paso 4**: Balancear hidrógenos con H⁺

**Reducción**: MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O
- Hay 8 hidrógenos en las 4 H₂O, necesitamos 8 H⁺ del lado izquierdo

**Paso 5**: Balancear las cargas con electrones

**Reducción**: 
- Lado izquierdo: -1 + 8(+1) = +7
- Lado derecho: +2
- Diferencia: necesitamos 5e⁻ del lado izquierdo

**MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O**

**Oxidación**:
- Lado izquierdo: +2
- Lado derecho: +3
- Diferencia: se pierde 1e⁻

**Fe²⁺ → Fe³⁺ + 1e⁻**

**Paso 6**: Igualar los electrones

Para que los electrones se cancelen, multiplicamos la oxidación por 5:

**Reducción**: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
**Oxidación × 5**: 5Fe²⁺ → 5Fe³⁺ + 5e⁻

**Paso 7**: Sumar ambas ecuaciones

**MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺**

**RESPUESTA FINAL**: 
**MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺**

### EJERCICIO 3: Balancear en medio básico

**Problema**: Balancea en medio básico:

**Cr₂O₇²⁻ + SO₃²⁻ → Cr³⁺ + SO₄²⁻**

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Balancear como si fuera medio ácido (pasos del ejercicio 2)

**Reducción**: Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
**Oxidación**: SO₃²⁻ + H₂O → SO₄²⁻ + 2H⁺ + 2e⁻

**Paso 2**: Igualar electrones (multiplicar oxidación por 3)

**Reducción**: Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
**Oxidación × 3**: 3SO₃²⁻ + 3H₂O → 3SO₄²⁻ + 6H⁺ + 6e⁻

**Paso 3**: Sumar

**Cr₂O₇²⁻ + 14H⁺ + 3SO₃²⁻ + 3H₂O → 2Cr³⁺ + 7H₂O + 3SO₄²⁻ + 6H⁺**

Simplificando:
**Cr₂O₇²⁻ + 8H⁺ + 3SO₃²⁻ → 2Cr³⁺ + 4H₂O + 3SO₄²⁻**

**Paso 4**: Convertir a medio básico (añadir OH⁻)

Añadimos 8OH⁻ a ambos lados para neutralizar los 8H⁺:

**Cr₂O₇²⁻ + 8H⁺ + 8OH⁻ + 3SO₃²⁻ → 2Cr³⁺ + 4H₂O + 3SO₄²⁻ + 8OH⁻**

Los H⁺ + OH⁻ forman H₂O:

**Cr₂O₇²⁻ + 8H₂O + 3SO₃²⁻ → 2Cr³⁺ + 4H₂O + 3SO₄²⁻ + 8OH⁻**

Simplificando el agua:

**RESPUESTA FINAL**:
**Cr₂O₇²⁻ + 4H₂O + 3SO₃²⁻ → 2Cr³⁺ + 3SO₄²⁻ + 8OH⁻**

## 2. PILA ELECTROQUÍMICA - Ejercicios

### EJERCICIO 4: Calcular potencial estándar de celda

**Problema**: Calcula el potencial estándar de una pila formada por:

- **Semi-reacción 1**: Zn²⁺ + 2e⁻ → Zn (E° = -0.76 V)
- **Semi-reacción 2**: Cu²⁺ + 2e⁻ → Cu (E° = +0.34 V)

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Identificar cuál es el cátodo y cuál es el ánodo

El que tiene **mayor E°** será el **CÁTODO** (se reduce):
- Cu²⁺ + 2e⁻ → Cu (E° = +0.34 V) → **CÁTODO**

El que tiene **menor E°** será el **ÁNODO** (se oxida):
- Zn²⁺ + 2e⁻ → Zn (E° = -0.76 V) → **ÁNODO**

**Paso 2**: Invertir la reacción del ánodo (oxidación)

**Ánodo**: Zn → Zn²⁺ + 2e⁻ (E° = -0.76 V)
⚠️ **IMPORTANTE**: El potencial NO cambia de signo al invertir

**Paso 3**: Calcular el potencial total

**ΔE° = E°(cátodo) - E°(ánodo)**

**ΔE° = (+0.34 V) - (-0.76 V)**
**ΔE° = +0.34 V + 0.76 V**
**ΔE° = +1.10 V**

**Paso 4**: Verificar espontaneidad

Como **ΔE° > 0** → La reacción es **ESPONTÁNEA** ✓

**RESPUESTA**: 
- Potencial de celda: **+1.10 V**
- La pila es espontánea y puede generar electricidad

**Reacción global**:
**Zn + Cu²⁺ → Zn²⁺ + Cu**

### EJERCICIO 5: Ecuación de Nernst (condiciones no estándar)

**Problema**: Calcula el potencial de la pila del ejercicio anterior a 25°C si:
- [Zn²⁺] = 0.10 M
- [Cu²⁺] = 2.0 M

Datos: E° = +1.10 V, n = 2

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Escribir la reacción y el cociente Q

**Reacción**: Zn + Cu²⁺ → Zn²⁺ + Cu

**Q = [Zn²⁺]/[Cu²⁺]** (los sólidos no se incluyen)

**Paso 2**: Calcular Q

**Q = 0.10/2.0 = 0.05**

**Paso 3**: Aplicar la ecuación de Nernst

**E = E° - (0.0592/n) × log Q**

**E = 1.10 - (0.0592/2) × log(0.05)**

**Paso 4**: Calcular el logaritmo

**log(0.05) = -1.301**

**E = 1.10 - (0.0296) × (-1.301)**

**E = 1.10 + 0.0385**

**E = 1.14 V**

**RESPUESTA**: 
El potencial de la celda en estas condiciones es **1.14 V**

**Interpretación**: El potencial aumentó porque hay menos [Zn²⁺] (producto) y más [Cu²⁺] (reactante), lo que favorece la reacción hacia los productos.

### EJERCICIO 6: Calcular ΔG a partir de E°

**Problema**: Para la pila Zn/Cu con E° = +1.10 V y n = 2, calcula ΔG°.

Dato: F = 96,485 C/mol

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Usar la fórmula

**ΔG° = -nFΔE°**

**Paso 2**: Sustituir valores

**ΔG° = -(2)(96,485 C/mol)(1.10 V)**

**ΔG° = -212,267 J/mol**

**ΔG° = -212.3 kJ/mol**

**RESPUESTA**: 
- ΔG° = **-212.3 kJ/mol**
- Como ΔG° < 0 → La reacción es **ESPONTÁNEA** ✓

## 3. SOLUBILIDAD Y Kps - Ejercicios

### EJERCICIO 7: Calcular Kps a partir de solubilidad

**Problema**: La solubilidad del AgCl en agua a 25°C es 1.3 × 10⁻⁵ M. Calcula el Kps.

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Escribir la ecuación de disolución

**AgCl(s) ⇌ Ag⁺(ac) + Cl⁻(ac)**

**Paso 2**: Hacer la tabla ICE (Inicial, Cambio, Equilibrio)

|  | AgCl(s) | Ag⁺ | Cl⁻ |
|---|---------|-----|-----|
| Inicial | - | 0 | 0 |
| Cambio | - | +s | +s |
| Equilibrio | - | s | s |

Donde s = solubilidad = 1.3 × 10⁻⁵ M

**Paso 3**: Escribir la expresión de Kps

**Kps = [Ag⁺][Cl⁻]**

**Paso 4**: Sustituir y calcular

**Kps = (s)(s) = s²**

**Kps = (1.3 × 10⁻⁵)²**

**Kps = 1.69 × 10⁻¹⁰**

**RESPUESTA**: **Kps = 1.7 × 10⁻¹⁰**

---

### EJERCICIO 8: Calcular solubilidad a partir de Kps

**Problema**: El Kps del PbCl₂ es 1.7 × 10⁻⁵ a 25°C. ¿Cuál es su solubilidad molar?

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Escribir la ecuación de disolución

**PbCl₂(s) ⇌ Pb²⁺(ac) + 2Cl⁻(ac)**

⚠️ **NOTA**: Observa que por cada Pb²⁺ se producen 2 Cl⁻

**Paso 2**: Tabla ICE

|  | PbCl₂(s) | Pb²⁺ | Cl⁻ |
|---|----------|------|-----|
| Inicial | - | 0 | 0 |
| Cambio | - | +s | +2s |
| Equilibrio | - | s | 2s |

**Paso 3**: Expresión de Kps

**Kps = [Pb²⁺][Cl⁻]²**

**Paso 4**: Sustituir y resolver

**1.7 × 10⁻⁵ = (s)(2s)²**

**1.7 × 10⁻⁵ = (s)(4s²)**

**1.7 × 10⁻⁵ = 4s³**

**s³ = (1.7 × 10⁻⁵)/4**

**s³ = 4.25 × 10⁻⁶**

**s = ³√(4.25 × 10⁻⁶)**

**s = 1.62 × 10⁻² M**

**RESPUESTA**: 
La solubilidad molar del PbCl₂ es **1.62 × 10⁻² M** o **0.0162 M**

---

### EJERCICIO 9: Predecir precipitación con Q

**Problema**: Se mezclan 100 mL de AgNO₃ 0.002 M con 100 mL de NaCl 0.001 M. 
¿Se formará precipitado de AgCl? (Kps AgCl = 1.7 × 10⁻¹⁰)

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Calcular concentraciones después de mezclar

Al mezclar, el volumen se duplica (200 mL total)

**[Ag⁺] = (0.002 M)(100 mL)/(200 mL) = 0.001 M**

**[Cl⁻] = (0.001 M)(100 mL)/(200 mL) = 0.0005 M**

**Paso 2**: Calcular Q (cociente de reacción)

**Q = [Ag⁺][Cl⁻]**

**Q = (0.001)(0.0005)**

**Q = 5 × 10⁻⁷**

**Paso 3**: Comparar Q con Kps

**Q = 5 × 10⁻⁷**
**Kps = 1.7 × 10⁻¹⁰**

**Q > Kps** → **SÍ precipita** ✓

**RESPUESTA**: 
Sí se formará precipitado de AgCl porque Q > Kps. El sistema está sobresaturado y debe precipitar para alcanzar el equilibrio.

### EJERCICIO 10: Efecto del ión común

**Problema**: ¿Cuál es la solubilidad del AgCl en una solución de NaCl 0.10 M?
(Kps AgCl = 1.7 × 10⁻¹⁰)

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Reconocer el ión común

El NaCl proporciona iones Cl⁻, que es común con el AgCl.

**Paso 2**: Ecuación de disolución

**AgCl(s) ⇌ Ag⁺(ac) + Cl⁻(ac)**

**Paso 3**: Tabla ICE (considerando el Cl⁻ inicial)

|  | Ag⁺ | Cl⁻ |
|---|-----|-----|
| Inicial | 0 | 0.10 M |
| Cambio | +s | +s |
| Equilibrio | s | 0.10 + s |

**Paso 4**: Aplicar Kps

**Kps = [Ag⁺][Cl⁻]**

**1.7 × 10⁻¹⁰ = (s)(0.10 + s)**

**Paso 5**: Simplificación

Como Kps es muy pequeño, s será muy pequeño comparado con 0.10:

**0.10 + s ≈ 0.10**

**1.7 × 10⁻¹⁰ = (s)(0.10)**

**s = (1.7 × 10⁻¹⁰)/0.10**

**s = 1.7 × 10⁻⁹ M**

**Paso 6**: Comparar con solubilidad en agua pura

En agua pura: s = 1.3 × 10⁻⁵ M
Con ión común: s = 1.7 × 10⁻⁹ M

**RESPUESTA**: 
La solubilidad del AgCl disminuye de **1.3 × 10⁻⁵ M** a **1.7 × 10⁻⁹ M** debido al efecto del ión común (¡es casi 10,000 veces menos soluble!)

## 4. TERMOQUÍMICA - Ejercicios

### EJERCICIO 11: Calcular calor específico

**Problema**: Se necesitan 450 J para elevar la temperatura de 25 g de una sustancia de 20°C a 30°C. Calcula el calor específico.

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Identificar datos

- q = 450 J
- m = 25 g
- Ti = 20°C
- Tf = 30°C
- Δt = Tf - Ti = 10°C

**Paso 2**: Usar la fórmula

**q = m × s × Δt**

Despejando s:

**s = q/(m × Δt)**

**Paso 3**: Sustituir valores

**s = 450 J/(25 g × 10°C)**

**s = 450/250**

**s = 1.8 J/g·°C**

**RESPUESTA**: 
El calor específico de la sustancia es **1.8 J/g·°C**

---

### EJERCICIO 12: Calcular ΔH con Ley de Hess

**Problema**: Calcula el ΔH de la reacción:

**C(s) + 2H₂(g) → CH₄(g)**

Usando estas reacciones:
1. C(s) + O₂(g) → CO₂(g)  ΔH₁ = -393.5 kJ
2. H₂(g) + ½O₂(g) → H₂O(l)  ΔH₂ = -285.8 kJ
3. CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)  ΔH₃ = -890.3 kJ

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Identificar qué necesitamos

Queremos: **C + 2H₂ → CH₄**

**Paso 2**: Manipular las ecuaciones dadas

**Ecuación 1**: Ya tiene C del lado izquierdo ✓
**C(s) + O₂(g) → CO₂(g)  ΔH₁ = -393.5 kJ**

**Ecuación 2**: Necesitamos 2H₂, así que multiplicamos por 2:
**2H₂(g) + O₂(g) → 2H₂O(l)  ΔH₂' = 2(-285.8) = -571.6 kJ**

**Ecuación 3**: Necesitamos CH₄ del lado derecho, así que la invertimos:
**CO₂(g) + 2H₂O(l) → CH₄(g) + 2O₂(g)  ΔH₃' = +890.3 kJ**

**Paso 3**: Sumar las ecuaciones

```
C(s) + O₂(g) → CO₂(g)                    ΔH = -393.5 kJ
2H₂(g) + O₂(g) → 2H₂O(l)                 ΔH = -571.6 kJ
CO₂(g) + 2H₂O(l) → CH₄(g) + 2O₂(g)       ΔH = +890.3 kJ
_______________________________________________________
C(s) + 2H₂(g) → CH₄(g)                   ΔH = ?
```

**Paso 4**: Cancelar términos comunes

- CO₂ aparece en ambos lados → se cancela
- 2H₂O aparece en ambos lados → se cancela
- 2O₂ aparecen en ambos lados → se cancelan

Queda: **C(s) + 2H₂(g) → CH₄(g)**

**Paso 5**: Sumar los ΔH

**ΔH = -393.5 + (-571.6) + 890.3**

**ΔH = -74.8 kJ**

**RESPUESTA**: 
**ΔH = -74.8 kJ** (exotérmico, libera calor)

### EJERCICIO 13: Calorimetría

**Problema**: Se mezclan 50 mL de HCl 1.0 M con 50 mL de NaOH 1.0 M en un calorímetro. La temperatura sube de 20.0°C a 26.5°C. Calcula el calor de neutralización.

Datos: densidad = 1.0 g/mL, calor específico = 4.18 J/g·°C

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Calcular la masa total de la solución

**Volumen total = 50 mL + 50 mL = 100 mL**

**Masa = volumen × densidad = 100 mL × 1.0 g/mL = 100 g**

**Paso 2**: Calcular el cambio de temperatura

**Δt = Tf - Ti = 26.5°C - 20.0°C = 6.5°C**

**Paso 3**: Calcular el calor absorbido por la solución

**q = m × s × Δt**

**q = 100 g × 4.18 J/g·°C × 6.5°C**

**q = 2,717 J = 2.72 kJ**

**Paso 4**: Calcular moles de reacción

**moles HCl = 1.0 M × 0.050 L = 0.050 mol**
**moles NaOH = 1.0 M × 0.050 L = 0.050 mol**

Reacción 1:1, así que reaccionan 0.050 mol

**Paso 5**: Calcular ΔH por mol

Como la reacción libera calor (temperatura sube), el ΔH es negativo:

**ΔH = -q/moles = -2.72 kJ/0.050 mol**

**ΔH = -54.4 kJ/mol**

**RESPUESTA**: 
El calor de neutralización es **-54.4 kJ/mol** (exotérmico)

### EJERCICIO 14: Predecir espontaneidad con ΔG

**Problema**: Para una reacción a 25°C (298 K):
- ΔH = -92 kJ
- ΔS = -198 J/K

¿Es espontánea?

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Convertir unidades (ambos deben estar en kJ o J)

**ΔH = -92 kJ = -92,000 J**
**ΔS = -198 J/K**

O mejor:
**ΔS = -198 J/K = -0.198 kJ/K**

**Paso 2**: Aplicar la ecuación de Gibbs

**ΔG = ΔH - TΔS**

**ΔG = -92 kJ - (298 K)(-0.198 kJ/K)**

**ΔG = -92 kJ + 59.0 kJ**

**ΔG = -33 kJ**

**Paso 3**: Interpretar

**ΔG < 0** → **ESPONTÁNEA** ✓

**RESPUESTA**: 
Sí es espontánea a 25°C porque ΔG = -33 kJ < 0

**Análisis**:
- ΔH < 0 favorece la espontaneidad (exotérmico)
- ΔS < 0 NO favorece la espontaneidad (disminuye el desorden)
- Pero el efecto entálpico es mayor, así que es espontánea

### EJERCICIO 15: Calcular trabajo de expansión

**Problema**: Un gas se expande de 2.0 L a 5.0 L contra una presión constante de 1.5 atm. Calcula el trabajo realizado.

Dato: 1 L·atm = 101.3 J

#### SOLUCIÓN PASO A PASO:

**Paso 1**: Calcular el cambio de volumen

**ΔV = Vf - Vi = 5.0 L - 2.0 L = 3.0 L**

**Paso 2**: Aplicar la fórmula del trabajo

**w = -PΔV**

**w = -(1.5 atm)(3.0 L)**

**w = -4.5 L·atm**

**Paso 3**: Convertir a Joules

**w = -4.5 L·atm × 101.3 J/(L·atm)**

**w = -455.9 J ≈ -456 J**

**RESPUESTA**: 
El trabajo es **-456 J**

**Interpretación**:
- El signo negativo indica que el sistema REALIZA trabajo sobre los alrededores
- El gas se expande, por lo que gasta energía empujando contra la presión externa

## 5. CINÉTICA QUÍMICA - Ejercicios

### EJERCICIO 16: Calcular velocidad de reacción

**Problema**: En la reacción: **2N₂O₅ → 4NO₂ + O₂**

La concentración de N₂O₅ disminuye de 0.50 M a 0.30 M en 200 segundos.

Calcula:
a) La velocidad de desaparición de N₂O₅
b) La velocidad de aparición de NO₂

#### SOLUCIÓN PASO A PASO:

**Parte a): Velocidad de desaparición de N₂O₅**

**Paso 1**: Usar la fórmula

**v = -Δ[N₂O₅]/Δt**

⚠️ El signo negativo se usa porque la concentración disminuye

**Paso 2**: Calcular el cambio de concentración

**Δ[N₂O₅] = [N₂O₅]f - [N₂O₅]i = 0.30 M - 0.50 M = -0.20 M**

**Paso 3**: Calcular la velocidad

**v = -(-0.20 M)/200 s**

**v = 0.20 M/200 s**

**v = 0.001 M/s = 1.0 × 10⁻³ M/s**

**Parte b): Velocidad de aparición de NO₂**

**Paso 1**: Usar la estequiometría

De la ecuación: **2N₂O₅ → 4NO₂**

Por cada
